    STREAM: A stream is a sequence of bytes which have source and destination. we need streams to control the input and output operation.To done I/O operation streams provides some predefine classes and objects. In C++ mainly we us two types of streams input stream and output stream.

    The first main class is IOS class and in this class we have another two sub classes, Like istream and ostream predefine classes. istream controls the input operation and ostream controls the output operations.To done this two operation at one place we hava another class called iostrem class. And the main thing is iostream is the combination of both istream and ostream classes.

    All this four predefine classes present under one class called iostream.h, iostream.h is a header file. To do i/o operation in our program we have to us this header file(#include < iostream.h >). presently we are using new version compilers like Dev C++ in this compiler we us #include< iostream > along with this we use using namespace std;
