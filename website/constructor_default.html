constructor with default values

We deal with this concept when we work with the parameterized constructor. When we define the constructor we pass some parametersa and we assign some default values, at the time of object creation the constructor also invoke and we pass some values. And the no.of parameters is equal to the no.of parameters we passed.

Here the point comes, if we miss any parameter value at the time of passing the value at object creation then automatically the default value is going to assigned by compiler to that parameter.

let's see one example program

#include< iostream >
using namespace std;
class test{
int a,b,c;
public:
test(int x=10, int y=20, int z=30){
a=x;
b=y;
c=z;
}
void show()
{
cout<<"a=" << a << endl; cout<<"b=" << b << endl; cout<<"c=" << c<< endl; cout<<"---------------------" << endl;
} };
int main()
{
test t(1,2,3);
t.show();
test t1(2);
t1.show();
return 0;
}


a=1
b=2
c=3
---------------------
a=2
b=20
c=30
