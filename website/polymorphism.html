polymorphism

Polymorphism by name it suggest thay poly means many and morphi means forms. It is a key concept in C++. Polymorphism means doing many operation or task by using the same name by passing different arguments or different number of paramerters.

There are two types of polymorphism

  • compile time polymorphism
  • runtime polymorphism

    • 1). compile time polymorphism
    These type of polymorphism is achived by function overloading and operator overloading.
    Function overloading means using the same name for different function by changing no.of paramerters or passing different paramerters in the same class.

    1). function overloading
    2). operator overloading
