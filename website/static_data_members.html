static data members

A class data members are declared as static data members inside class by using the static keyword.
And this datamjembers are initilised with zero automatically.

let's have a talk about static data members, If suppose a class have a data member int a . And if we create a multilpe objects to that class then each object have its own local variable int a, if we assign a value to that variable it doesn't effect on another object variable. Because the different object variable have different memory location.

But when we declare a variable with static keyword, only one memory location is created and ii is shared to all the objects. If we change the value of that variable in one object then it will reflect or change the value in other object also.

When we declare a variable static, then it will becomes a class member not a object member and only one copy of that memory is createdand shared to all the objects. Static data members are access by both class and object. We must declare the static variable in inside the class and we have to define it outside the class.
