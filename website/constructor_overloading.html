constructor_overloading



constructor_overloading: Before we know about constructor overloading, first we have to know what is overloading?
overloading means defining the several functions or objects with same name by changing no.of arguments or changing the order of arguments within one class.
constructor overloading: Defining the two or more constructor with in the same class by changing the no.of parameters or changing the order of parameters. And it is used to extends the functionality of constructor.It is a constructor polymorphims.

#include< iostream>
#include < iostream>
using namespace std;
class rect{
int l,b;
public:
rect(){
l=20;
b=50;
}
rect(int x, int y){
l=x;
b=y;
}
void showarea()
{
cout<<" Area of rectangle:" << l*b << endl;
}
};
int main()
{
rect r;
r.showarea();
rect r1(30, 60);
r1.showarea();
return 0;
}
