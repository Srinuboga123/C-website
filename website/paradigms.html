Programming paradigms:-

Paradigms can also be termed as method/stategy to solve some problems or do some task.programming paradigm is an approach to solve a problem using tools and techniques that are available to us following some approach. There are alot of programming language that are known but all of them need to follow some stategy when they are implemented and this methodology stategy is paradigms.

THE PROGRAMMING PARADIGMS ARE OF TWO TYPES:
1.Imperative programming paradigms
 2.Declarative programming paradigms
