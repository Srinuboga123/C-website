multilevel_inheritance


multilevel inheritance, let's take a general example for better understand suppose we have 4 classes A,B,C,D. if we consider A as base class and then we derived a another class B from the A class, again we derived another class C from the class B, now the class B becames the parent class to the C class. Here we get all the properties of base class, intermediate base class to the last derived class.
Now we say that B class is the intermediate base class.

Multiple inheritance contains the one base class and intermediate classes and one derived class. Here we passed all the properties of base class and intermediate classes to the derived class.

class abc{
int id;
char name[10];

int m1,m2,m3;

};
class average : public xyz{
int total, avg;

void sum(){
total=m1+m2+m3; cout<<"total=" << total<< endl;
}
void avg(){
if(total>=40){
cout<<"you pass:" << endl;
}
else{
cout<<"sorry! you not passed"<< endl;
}
avg=total/3;
cout<<"average=" << avg << endl;
}
};
int main(){
average a;
a.getd();
a.putd();
a.put();
a.sum();

}
