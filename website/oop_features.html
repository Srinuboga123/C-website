    3.encapsulation

    The process of binding the data members and member functions of a class into single unit is called encapsulation .

    example for encapsulation

    class employee
    {
    int empid;
    char empname[20];
    public:
    void get();
    void put();
    };

    In the above program int and char are the data members , get() and put() are the member funcation, we bind this two members into single unit by using class ,so we can encapsulate our data by using class.
