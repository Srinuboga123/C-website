bitwise operators

The operators which are used to work with bits and perform bit-by-bit operation. These operators are (&,|,^,~,<<,>>)


First the bitwise operators convert the numericaol value into binary digits/bits and perform the operation on each bit are return the result.
If suppose a=10 and b=0, by using bitwise operators(&), first the value is converted into bits 1010 and b value 0 is converted into 0000 bits. Then it perform (a&b) operation and returns the result is 0000.

  • & =binary AND operator
  • | = binary OR operator
  • ^ = binary XOR operator
  • ~ = binary ones complement operator
  • <<= binary left shift operator
  • >>= binary rigth shift operator
