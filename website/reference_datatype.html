reference datatype


Reference datatype is anew concept in c++.Reference is a new datatype and it is derived datatype.
In c we use pointers, array and functions in along with these in c++ we use another new derived dataype called reference dataype/ variable.

In c function we pass the parameters in two ways call by value and call by address along with thses in c++ we are going to use pass parameters by call by reference technique.

In functions we use pointers, pointer store any variable address of same type at anytime in future and we can change the address of with already stored in it further.Due to these pointer is not secured, To overcome these problem they introduce the reference variable. Reference variables are also called as link variables.

Reference variables allows the user to provide alias names for existing variable. Reference variables are mainly usedful in copy constructor and operator overloading

  • Reference variables are implicit constant vaeriables
  • Till now we know that & is address operator. But in c++ it is used for identify reference variable
    For example: int &a;

    • syntax for reference variable


    datatype &variable = variable;

    Important points about reference
  • Every rederence variable is initialize with another variable.
  • Reference variable should be initialized at the time of declaration only.

    • let's see one example:
    int a=10, &b=a;
    Just before we talk reference variables are linked variables. But how let's see
    Suppose int a=10, & b=a;
    in above example i take a=10 if i change the value of a to 20 and execute thet line cout<< a << b;, we get the output as 20 and 20, it's means here a=20 and b=20 means if we change anyone variable value then it's change that reference variable also.

    example program
    #include < iostream>
    using namespace std;
    int main()
    {
    int a=10;
    int &b=a;
    int &c=b;
    cout<< "Before the values change"<< endl;
    cout<<"a:"<< a<< endl;
    cout<<"b:"<< b<< endl;
    cout<<"c:"<< c<< endl;
    a=100;
    cout<<"After the value change"<< endl;
    cout<<"a:"<< a<< endl;
    cout<<"b:"<< b<< endl;
    cout<<"c:"<< c<< endl;
    return 0;
    }
    output
    Before the values change
    a:10
    b:10
    c:10
    After the value change
    a:100
    b:100
    c:100

    Reference variable allows ti share the data between function. In c to share the data in function we use call by address technique, But in c++ we are using call by reference method.

    #include < iostream>
    using namespace std;
    int main()
    {
    int y=200, z=400;
    cout<<"Before swapping"<< endl;
    cout<<"y="<< y<< endl;
    cout<<"z="<< z<< endl;
    swap(y,z);
    cout<<"After swapping"<< endl;
    cout<<"y="<< y<< endl;
    cout<<"z="<< z<< endl;
    return 0;
    }
    void swap(int &y, int &z)
    {
    int temp=y;
    y=z;
    z=temp;
    }
    output
    Before swapping
    y=200
    z=400
    After swapping
    y=400
    z= 200
