assignment operators

5)Assignment operators
The operators which are used to assign values of rigth side of assignment operator to left side operand of an assignment operator.THE left side of operand of operator is called variable.
The datatype on both side of operator should be same type . otherwise it will rise an error , compile time error.
The assignment operators are(=,+=,-=,*=,/=,%=,<<=,>>=,&=,^=,!=)

example program for assignment operators
#include< iostream >
using namespace std;
int main()
{
int x=5;
cout<< "initial value of x is:" << x << endl;
x+=5;
cout<< "x+=5 gives:" << x << endl;
x-=5;
cout<< "x-=5 gives:" << x << endl;
x*=5;
cout<< "x*=5 gives:" << x << endl;
x/=5;
cout<< "x/=5 gives:" << x << endl;
return 0;
}
x+=5 gives : 10
x-=5 gives : 10
x*=5 gives : 10
x/=5 gives : 10
