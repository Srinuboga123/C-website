operators in c++


Operators is a symbol that tells the compiler to perform specific operation.Every operator have its own predefined operation which are already predefined by the developers.And c++ is rich with built-in operators.

there are 6 types of operators in c++

  1. Arithmetic operators
  2. relational operators
  3. logical operators
  4. Bitwise operators
  5. Assignment operators
  6. Misc operators

1)arithmetic operators
The operators which are used for performing mathematical operations like addition, subtraction, multiplication,division, increment and decrement operations.
Arithmetic operators are (+,-,*,/,%,++,--)

Arithmetic operators are of two types:
a) unary operators
b). Binary operators

Unary operators :- The opeartors which perform opeart with single operand only. There are (--,++)
Binary operators :- The operators which perform operation with teo operands are called binary operators. Those are(+,-,*,/,%)

2)relational operators
The operators which are used to know the relation between operands or to compare two operands.
the relation operators are (==,!=, < ,>,>=,<=)

3)logical operators
The operators which are used to check condition of two expression, if the condition is true it returns 1, if condition is false it return 0. here the operands may be non-zero or zero.
The logical operators are (&&,||,!)

  • && is called logical AND operator
  • || is called logical OR operator
  • ! is logical NOT
    • for exmaple program click here example_program
    For example:- let's take two operands values a=0and b=1
  • && is called logical AND operator, it return the true value when both values are non-zero only.
  • || is called logical OR operator, it returns the false value when the both operands values are zero,And return true value if both the values are non-zero/ anyone of the operand is zero.
  • ! is a logical NOT operator, if condition is true it returns false, if it's false it return true.


    • 4)Bitwise operators
    The operators which are used to work with bits and perform bit-by-bit operation. These operators are (&,|,^,~,<<,>>)

    First the bitwise operators convert the numericaol value into binary digits/bits and perform the operation on each bit are return the result.
    If suppose a=10 and b=0, by using bitwise operators(&), first the value is converted into bits 1010 and b value 0 is converted into 0000 bits. Then it perform (a&b) operation and returns the result is 0000.

  • & =binary AND operator
  • | = binary OR operator
  • ^ = binary XOR operator
  • ~ = binary ones complement operator
  • <<= binary left shift operator
  • >>= binary rigth shift operator
