generics/templets

What is generics? Before knowing about generics, let's hava brife discussion about funtion overloading. We know that by using function overloading we can extended the funcationality of the function by using the same function name passing different parameters. By using function overloading we write code for different task, by doing this the code of the program is increases.

Based on this to decrease the code of a function without decreasing the performance and activity. Here the generics come into the picture.
Generics are nothing but templates, generics is concept with is used to performe many operation by using single class or function. To performe operations with generics we use the templete keyword here we use one placeholder to performe operation. And in generics function we must have to pass the arguments. Generics are of two types
1). generics function
2). generics class
