    • cout

    Cout is a predefine object with is available in ostream class. To perform operations with cout first we need to use the iostream.h header file in our program.

    let's see some features of cout,

    • 1) cout always refers to the console output device i.e monitor.In simple words we use cout to display the content on the screen.

    • 2). We can perform the formatted and unformatted output operations with cout object.

    • 3). Another advantage feature in c++ is cout doesn't required any conversion character like %d,%s,%f with we have us in c language. Because implicit conversion is done in cout object, that's why we don't use conversion character in c++ programming.

    Syntax for cout object



    cout<<"text/variable";

    Above syntax we use the "<<" operator, we called it as a insertion operator/put to operator and we called it as over loaded left shift opeartor.
    cout<<"welcome to c++ world";
    int a=3;
    int b=34;
    cout<< a;
    cout<< b ;

    cin

    Another important predefine object provided by the istream class is cin. Mostly we use cin object to give input data to our program. Cin always refer standard input device i.e keybord,it always take the input from keyboard. Cin is available in istream predefine class and we use iostream.h headerfile to perform cin operation, how we use headerfile for cout.

    syntax for cin:
    cin>>variable>>variable>>...;

    In the above syntax we use ">>" we call it as extraction operator and we also called it as over loaded rigth shift operator.

    cin doesn't required address operator. Because implicit conversion is done.
    Example:
    int a;
    int b;
    cin>> a;
    cin>> b;
    we can give two inputs by using single cin object, Like
    cin>>a>>b;
    here we use extraction operator in two times its refers to cascading.

    program 1

    #include< iostream >
    using namespace std;
    int main()
    {
    int a;
    int b;
    cout<<"enter a and b values"<< endl;
    cin>>a;
    cin>>b;
    cout<<"a="<< a<< endl;
    cout<< "b="<< b<< endl;
    return 0;
    }

    output


    enter a and b values
    24
    100
    a=24
    b=100
