logical operators

The operators which are used to check condition of two expression, if the condition is true it returns 1, if condition is false it return 0. here the operands may be non-zero or zero.
The logical operators are (&&,||,!)

  • && is called logical AND operator
  • || is called logical OR operator
  • ! is logical NOT
    • For example:- let's take two operands values a=0and b=1
  • && is called logical AND operator, it return the true value when both values are non-zero only.
  • || is called logical OR operator, it returns the false value when the both operands values are zero,And return true value if both the values are non-zero/ anyone of the operand is zero.
  • ! is a logical NOT operator, if condition is true it returns false, if it's false it return true.
