data types in c++ :-

variable:-

A variable is used for storing either a character or value.Each variable in c++ have specific type which can be determine the size of the variable in memory.And each variable need identifier to distinguish it from other variable. Variable names in c++ can be write upto from 1 to 225 character. All variables names must begin with a letter of alphabet or an underscore. While we writing variables no spaces or special characters are allowed and the important thimg is c++ is a case-sensitive uppercase or lowercase letters are distinguish.

identifiers:-

An identifier is a sequence of muitiple lettes digits or underscore character(_). Neither spaces nor punctuation marks or symbols can be part of an identifier. We have to use only letters,digits and single underscore character then only it can vaild.

Another important point is while we inventing our own identifier it does not match eith any keywords of c++ language, if not the complier specifics one which are reserved keywords.


The c++ standard reserved ketwords are,

auto,break,case,catch,try,do,while,class, default,int,char,float,double,if,else,switch, new,throw, typedef, union, unsigned, singed, virtual, friend, public, privite, short, return, oparator, templete, typename, inline, long, const,using, protected,for,namespace, nested if, register, extern, sizeof, false, this, static,cast, etc..

In c++ there 3 types of data types

  1. 1).Primitive data type
  2. 2).Derived data types
  3. 3).User-defined data types

1.Primitive data types:-

we can also called it as built-in data types. These datya types are predefined and user can be use directly by declaring.
There are some primitive data types available in C++

a).Integer
b).Character
c).Floating point
d).Double
e).Boolean
f).void
g).wide character


2. Derived data types:-

These data types are derived from the primitive data types.Primitive datatypes are reffered to as derived datatypes.
The derived datatypes are

a).Function
b).Array
c).pointer
d).reference

3. User defined data types:-


User defined datatypes are basically derived from the built-in datatypes and these are customized as user defined datatypes.

There user defined datatypes are

a).Structure
b).Union
c).Enumaration
d).Class

In this page we discuss about primitive data types
a).character:-
Character is a primitive datatype. These datatype is used for storing characters. we use char keyword to declare character datatype. The size of memeory required for char is 1 byte.
For example:- char a;

b). Integer:-
It ia a primitive datatype it is used for storing the integer number. We use int keyword for declaring the integer datatype.It required 4 bytes of memory

For Example:-int a;

C).Boolean:-
It is a primitive datatype used for storing logical values(T OR F). We use bool keyword for declaring boolean datatype.The size of memory required for boolean is 1 byte.Range is 0 to 1.
we can not have use modifiers for this datatype.

d).Floating point:-
Floating point is a primitive datatype and it is used for storing single precision value. We use float keyword for declaring float datatype. It required 8 bytes if memory.
for example:-float a;

e). double floating point:-
This datatype is a primitive datatype. it is used for storing double precision values. We use double keyword to declare it.It required 8 bytes of memeory
for example:- double as;

f).void:-
Void means without any value. void datatypes represents a valueless entity. Void datatype is uses in function which does not return a value. it doesn't required memory.

for example:- void v;


sizeof() is a built-in function used for finding the sizeof datatype.

#include< iostream>
using namespace std;
int main()
{
int a;
cout<<"size of integer(int):"<< sizeof(a)<< endl;
char c;
cout<<"size of character(char):"<< sizeof(c)<< endl;
float f;
cout<<"size of float(float):"<< sizeof(f)<< endl;
double d;
cout<<"size of double(double):"<< sizeof(d)<< endl;
return 0
}
output;-
size of integer(int):-4
size of character(char):-1
size of float(float):-4
size of double(double):-8
