inheritance



Inheritance is one of the important concept in oops. Inheritance is the process of deriving a new class from the existing class. In other words, The process of passing the super class properties to the derived class.
Inheritance allowes the code reusability and extensibility.And it increase the performance of the program.

syntax for inheritance

class derivedClassName : public/private/protected baseclass
{
};
  • Here we use the visibility mood, this mood tell the behaviour of the base class members function in the derived class.
  • Wecan derive classes from the multiple classes or muitiple paths.
  • Let's have a look what happen when inheritance is done. The memory is allocated to the derived class and base class when derived class object is created.
  • The member availability in derived class is based on derivation mood or visibility mood.
    • Behaviour of member based on visibility mood
  • When visibility mood of a derived class is public, then public member of base class is one again public member of derived class, And the protected member of of base class are protected in the derived class also.
  • If the derived class visibility mood is private, then the public mood of base class are private members in the derived class. And the protected members of the base also become the private members in derived class.
  • The private member of the base class are not available in the derived class if the visibility mood is public/private. The private member are not partcipate in inheritance because of data hiding.
