constructor

Constructor is a new concept is introduce in c++. What is constructor?

Constructor is a special member function unlike normal function, it as all the featutes of normal function alone with it's own features, except, every normal function hava return data type like(int float void , etc). But the constructor doesn't have the any return data type includeing void data type also. Normally in c language we can able to assign the value to the data member at the time of declaration. But in c++ we can't assign the value to the data members at the time of declaration. Because the data members are in private mood by default, and moreover see keep our data members in private to achive the data hiding, the main aim of the c++ is data hiding. We use the Constructor to assign value to data members, c++ allows this property. The constructor is calles by compiler automatically when the object is created to the class.

Rules for constructor
  1. constructor name should be same as the class name.
  2. It dosen't return any return value, and it doesn't hava any return prototype like(int, float, void).
  3. we have to declare the constructorin public mood only , like normal member function.
There are three types of constructors:
  • Default constructor
  • parameterized constructor
  • copy constructor
