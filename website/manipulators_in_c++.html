    • manipulators


    Another main concept in c++ is manipulators, manipulators are used for the formatted output purpose. To perform the operations with manipulators we need to use the < iomanip.h > header file.
    Here we have two types of manipulators:
    1). manipulator operators
    2).manipulator functions

    • 1).Here we have to discuss some more points, while we performming operations with manipulator operators we need to use < iomanip.h > header file, but it is optional.
    • 2). But while we performming operations with manipulators functions we have to implement the iomanip.h header file.
    • 3).All the manipulators works on the cout object. we know that cout is available in iostream.h header file.But if we implement iomanip.h header file in our program there is not need to use iostream.h header file. Because already the iomanip.h have the inner function of cout.
    • 4).This is only for cout, for cin object we have to use the iostream.h header file

    1).manipulator operators: The operators which are used to get the formatted result on display is called manipulator operators.

    This operators are of two types:
    A). endl
    B).ends

    A) endl:The endl operator is use to print the output in different new lines.In general we use "\n" to print the output in different new line. Then why we need to use use endl opeartor insteade of "\n", Here the point we know that "\n" is a backslash character and for every backslash i.e \n it takes one byte of memeory.But the endl operator is a manipulator and it doesn't required any memory.

    program 1
    #include< iostream >
    using namespace std;
    int main()
    {
    cout<<"srinu CSE\n";
    cout<< "GNIT";
    return 0;
    }

    srinu CSE
    GNIT

    B) ends: This operator is used for end space. lets have a clear idea about ends.
    Take one example cout<<"ab"<<"bc";, if we execute line we get the output abcd.But we want the space between ab and cd Like "ab cd", to get our desire output we need to use ends operator cout<<"ab"<< ends<<"cd"; .And one more thing ends give only one space. In c we use tab space "\t" to get two or more space.



    1).manipulator functions:

    The functions which are used to get formatted output are called manipulator functions. let's have a look on over it .


    A).setw(): setw() is a manipulator function, by name we can understand that setw means set width, setw() function going to allocate specified number of spaces on the screen and it print the text from right to left. Generally, the printf() and cout print the text from left to right.

    let's have a general example, Generally when we go to shopping after the payment we get the bill, on the bill content is in formatted it's means, the bill contain proper number of rows and coloum. Usually the item name is on left side and amount is on right side, once observe amount format is from right to left then only it's possible to calculate the amount.



    Example program by using \n


    #include< iostream>
    #include< iomanip>
    int main()
    {
    cout<<"2\n 22\n 222\n";
    return 0;
    }
    output
    2
    22
    222

    Example program by using setw()


    #include< iostream>
    #include< iomanip>
    int main()
    {
    cout<< setw(5)<< 2<< endl;
    cout<< setw(5)<< 2<< endl;
    cout<< setw(5) << 2<< endl;
    return 0;
    }
    output
    _ _ _ _2
    _ _ _ 22
    _ _ 222
    Here we give 5 as a parameter to get 5 spaces

    B). setfill(): setfill() is one of the function in manipulator functions. There is nothing much to discuss about setfill() function. The function of setfill is to fill empty spaces with are left by setw() function. Suppose if we use only 2 spaces of setw() out of 5 spaces then the remaining spaces will be filled by any symbol.

    And one more important point is setfill() and setw() are interrelaled without setw() setfill() function does not perform it's own operation.


    #include< iostream>
    #include< iomanip>
    int main()
    {
    cout<< setfill('*');
    cout<< setw(5)<< 2<< endl;
    cout<< setfill('*');
    cout<< setw(5)<< 2<< endl;
    cout<< setfill('*');
    cout<< setw(5) << 2<< endl;
    return 0;
    }
    output
    ****2
    ***22
    **222

    C). setprecision(): setprecision() is a manipulator function, the function of setprecision() is to specific number of decimal value after the decimail point in floating data and it doesn't works on integer data.
