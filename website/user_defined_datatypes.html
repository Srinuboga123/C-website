user defined datatypes


User defined datatypes is a datatype in c++, in which the data is defined by user.
User defined datatypes are of 4 types
1).Structure
2).Union
3).enumaration
4).Class

1).Structure
Structure is a user defined datatype in c++ and also in c. Structure is collection of different datatypes at one place under single name. And we use struct keyword for declaration of structure.we can access the structure data memebers by using structure variable followed by Dot(.) operator. We will learn more about structure in further artical.Click here to learn about structure


syntax for structure

struct structure_Name{
data member;
data member;
} structure_variable;


2).Union
The declaration and usage of union is same as structure. But the functionality of union is different. Here we use union keyword to declare it. Union members share same memory location and the size of union is equal to the size of the larger datatype. Addresses of the data members are same because of sharing same memory. we can access the union data memebers in by using union variable followed by Dot(.) operator.

syntax for union

union union_Name{
data member;
data member;
} union_variable;
#include < iostream>
using namespace std;
union a{
int a;
float b;
}u1;
int main()
{
cout<<"enter a value:"<< endl;
cin>>u1.a;
cout<<"address of a:"<<&u1.a<< endl;
cout<<"enter b value:"<< endl;
cin>>u1.b;
cout<<"address of b:"<<&u1.b<< endl;
cout<<"size of a :"<< sizeof(u1.a)<< endl;
cout<<"size of b:"<< sizeof(u1.b)<< endl;
return 0;
}
output:
enter a value:
23
address of a: 06012d4
enter b value:
34
address of b:06012d4
size of a: 4
size of b: 4
This output is get when i used the 16 bit compiler

3).Enumaration
Enumaration is a user defined datatype in c/c++. In enumaration we assign the intergral constant value to a name. Here we use enum keyword for declaration.

syntax for structure

nine, ten};

In the above program we didn't initilized any value to the names in the enum list. But when we excute the program we get some values, like zero = 0 and ten =9 by default. It means when we not initialized the value to the names, then the compiler will initialize some default values with start from 0 to n-1; where n is the number of elements in the list.

Suppose if we initialize some values to the names like zero=2 or one=3, if we run the program now get the zero=2 and one=3 and by continous we get two=4 and soon, it means if we give any value to a name in the list the compiler will give next value of the initialized value to the next name, like two=4 we given and for four the compiler initialize 5 . Let's see one example program for more understand

#include < iostream>
using namespace std;
enum numbers_alphabat{ zero=2, one=3 ,two ,three,four, five,six,seven,eigth, nine, ten};

int main()
{
printf("%d %d %d %d %d", zero ,one, two,three,four);
return 0;
}
2 3 4 5 6
